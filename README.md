# 智能校园助手项目说明

## 项目简介

智能校园助手是一款面向校园师生的综合服务平台，整合了课程管理、校园点餐、二手交易、积分系统等多种功能，旨在提供便捷的校园生活体验。

## 项目结构

```
smart-campus-assistant/
├── backend/            # 后端代码（Django）
├── frontend/           # 前端代码（Vue 2）
└── 需求分析.md        # 需求分析文档
```

## 技术栈

### 前端
- **框架**: Vue 2
- **UI组件库**: Element UI
- **状态管理**: Vuex
- **路由**: Vue Router
- **HTTP请求**: Axios

### 后端
- **框架**: Django 3.2
- **API框架**: Django REST Framework
- **数据库**: SQLite
- **认证**: JWT

## 快速开始

### 1. 启动后端服务

```bash
# 进入后端目录
cd backend

# 启动开发服务器
python manage.py runserver
```

后端服务将运行在：http://127.0.0.1:8000/

### 2. 启动前端服务

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run serve
```

前端服务将运行在：http://localhost:8080/

## 功能测试指南

### 1. 课程提醒功能测试

#### 方法一：等待实际时间触发
1. 登录系统
2. 进入"课程表"页面，确保有当天的课程
3. 在课程结束前1小时，系统会自动发送点餐提醒

#### 方法二：使用调试功能
1. 登录系统
2. 进入"设置"页面
3. 点击"调试设置"标签
4. 选择要模拟的时间（课程结束前1小时）
5. 点击"应用模拟时间"
6. 等待1分钟，系统会发送点餐提醒
7. 或直接点击"立即触发提醒"测试

### 2. 饮食推荐测试

1. 登录系统
2. 进入"餐厅列表"页面
3. 查看"今日推荐"和"性价比推荐"
4. 尝试搜索特定食材（如"米饭"、"牛肉"）
5. 点击"立即点餐"查看餐厅菜品

### 3. 积分系统测试

1. 登录系统
2. 进入"设置"页面
3. 点击"积分设置"标签
4. 查看积分获取规则
5. 点击"立即兑换"测试积分兑换功能

### 4. 校园生活大数据看板测试

1. 登录系统
2. 进入"校园生活数据看板"页面
3. 查看各项统计数据
4. 切换时间范围（本周/本月/本学期）
5. 查看学习-生活平衡情况

### 5. 统一通知中心测试

1. 登录系统
2. 点击顶部的铃铛图标
3. 查看所有通知
4. 切换"全部"、"未读"、"分类"视图
5. 点击通知进行操作

## 常见问题

### 1. 登录失败
- 检查默认账号密码是否正确
- 确保后端服务正在运行
- 检查浏览器控制台是否有错误

### 2. 没有收到课程提醒
- 检查"设置"页面的通知设置是否开启
- 检查课程表是否有当天的课程
- 使用调试功能测试提醒

### 3. 前端页面加载失败
- 确保前端服务正在运行
- 检查浏览器控制台是否有JS错误
- 尝试刷新页面

### 4. 后端API调用失败
- 确保后端服务正在运行
- 检查URL是否正确
- 查看后端控制台的错误日志
