# 智慧校园助手 - 用户手册

## 1. 系统简介

智慧校园助手是一个集成了二手交易、餐饮订单、课程表管理和实时聊天功能的校园综合服务平台，旨在为学生提供便捷的校园生活服务。

### 1.1 主要功能
- **二手交易**：发布、浏览、购买二手商品
- **餐饮订单**：在线点餐、查看订单
- **课程表**：管理个人课程表
- **实时聊天**：与其他用户进行实时沟通

### 1.2 技术特点
- 前后端分离架构，响应迅速
- 现代化的UI设计，操作简单
- 安全可靠的认证机制
- 跨平台支持，适配多种设备

## 2. 注册与登录

### 2.1 用户注册
1. 打开智慧校园助手应用
2. 点击"注册"按钮
3. 填写注册信息：
   - 用户名：设置您的账号名称
   - 密码：设置登录密码，建议包含字母、数字和特殊字符
   - 邮箱：用于找回密码和接收通知
   - 学号：您的学生学号
   - 手机号：用于接收验证码和通知
4. 点击"注册"按钮完成注册

### 2.2 用户登录
1. 打开智慧校园助手应用
2. 点击"登录"按钮
3. 输入您的用户名和密码
4. 点击"登录"按钮进入系统

### 2.3 忘记密码
1. 在登录页面点击"忘记密码"
2. 输入您的注册邮箱
3. 查收邮箱中的密码重置链接
4. 点击链接重置密码

## 3. 二手交易模块

### 3.1 商品浏览
1. 点击左侧菜单栏中的"二手交易"
2. 在商品列表页面查看所有在售商品
3. 可以通过顶部的分类筛选器筛选商品
4. 点击任意商品查看详情

### 3.2 商品详情
1. 在商品列表页面点击任意商品
2. 查看商品详细信息：
   - 商品名称、描述、价格
   - 商品图片
   - 发布者信息
   - 联系方式
3. 如果您不是商品发布者，可以点击"联系卖家"按钮与发布者沟通
4. 点击"收藏"按钮收藏商品
5. 在底部输入框中可以留言咨询

### 3.3 发布商品
1. 点击"发布商品"按钮
2. 填写商品信息：
   - 商品名称：简洁明了的商品名称
   - 商品描述：详细描述商品的成色、使用情况等
   - 商品价格：设置合理的售价
   - 商品分类：选择合适的分类
   - 商品图片：上传清晰的商品照片
   - 联系方式：方便买家联系您
3. 点击"发布"按钮完成商品发布

### 3.4 我的商品
1. 点击"我的商品"查看您发布的所有商品
2. 可以对商品进行编辑或删除操作
3. 可以查看商品的状态（在售/已售出）

### 3.5 我的收藏
1. 点击"我的收藏"查看您收藏的所有商品
2. 可以取消收藏商品
3. 点击商品可以查看详情

### 3.6 交易消息
1. 点击"交易消息"查看与商品相关的聊天消息
2. 显示所有对话列表，包含对方信息和最后一条消息
3. 点击对话进入聊天详情页

## 4. 聊天模块

### 4.1 对话列表
1. 点击左侧菜单栏中的"聊天"
2. 查看所有与您相关的对话
3. 每个对话显示对方的用户名和最后一条消息
4. 未读消息会显示红色提示

### 4.2 聊天详情
1. 在对话列表中点击任意对话
2. 进入聊天详情页，查看历史聊天记录
3. 消息显示格式：
   - 时间显示在消息气泡上方
   - 您发送的消息显示在右侧
   - 对方发送的消息显示在左侧
4. 在底部输入框中输入消息内容
5. 点击"发送"按钮发送消息

### 4.3 发起聊天
1. 在商品详情页点击"联系卖家"按钮
2. 系统会自动创建与商品发布者的对话
3. 进入聊天详情页，开始与对方沟通

## 5. 餐饮订单模块

### 5.1 餐厅浏览
1. 点击左侧菜单栏中的"餐饮订单"
2. 查看所有合作餐厅
3. 可以通过搜索框搜索特定餐厅

### 5.2 菜品浏览
1. 在餐厅列表中点击任意餐厅
2. 查看该餐厅的所有菜品
3. 每个菜品显示名称、价格和图片
4. 可以查看菜品的详细描述

### 5.3 购物车管理
1. 在菜品列表中点击"加入购物车"按钮
2. 点击右上角的购物车图标查看购物车
3. 可以修改菜品数量或删除菜品
4. 点击"结算"按钮进入订单确认页面

### 5.4 订单创建
1. 在购物车页面点击"结算"按钮
2. 确认订单信息：
   - 订单金额
   - 配送地址
   - 联系电话
   - 订单备注
3. 选择支付方式
4. 点击"提交订单"按钮完成订单创建

### 5.5 订单管理
1. 点击"我的订单"查看所有历史订单
2. 可以查看订单状态：
   - 待支付
   - 待接单
   - 制作中
   - 配送中
   - 已完成
   - 已取消
3. 点击订单可以查看订单详情

## 6. 课程表模块

### 6.1 课程表浏览
1. 点击左侧菜单栏中的"课程表"
2. 查看本周的课程安排
3. 可以切换周视图和日视图

### 6.2 添加课程
1. 点击"添加课程"按钮
2. 填写课程信息：
   - 课程名称
   - 教师姓名
   - 上课地点
   - 开始时间和结束时间
   - 星期几
3. 点击"保存"按钮完成课程添加

### 6.3 编辑课程
1. 在课程表中点击任意课程
2. 修改课程信息
3. 点击"保存"按钮完成修改

### 6.4 删除课程
1. 在课程表中点击任意课程
2. 点击"删除"按钮
3. 确认删除操作

## 7. 个人中心

### 7.1 个人信息
1. 点击右上角的头像进入个人中心
2. 查看个人基本信息
3. 点击"编辑"按钮修改个人信息

### 7.2 安全设置
1. 在个人中心点击"安全设置"
2. 可以修改密码
3. 可以绑定或解绑手机号和邮箱

### 7.3 通知设置
1. 在个人中心点击"通知设置"
2. 可以开启或关闭各类通知

## 8. 常见问题

### 8.1 如何联系客服？
- 在个人中心点击"联系客服"
- 发送消息描述您的问题
- 客服会在24小时内回复

### 8.2 如何修改商品信息？
- 进入"我的商品"
- 点击需要修改的商品
- 点击"编辑"按钮修改信息
- 点击"保存"按钮完成修改

### 8.3 如何取消订单？
- 进入"我的订单"
- 找到需要取消的订单
- 点击"取消订单"按钮
- 选择取消原因
- 点击"确认"按钮完成取消

### 8.4 如何评价交易？
- 交易完成后，进入"我的订单"
- 找到已完成的订单
- 点击"评价"按钮
- 填写评价内容和评分
- 点击"提交"按钮完成评价

### 8.5 如何查看聊天记录？
- 进入"聊天"模块
- 在对话列表中点击任意对话
- 查看完整的聊天记录

## 9. 注意事项

1. 请保护好您的账号和密码，不要泄露给他人
2. 发布商品时，请确保信息真实有效，避免欺诈行为
3. 交易时建议选择安全的交易方式，如面对面交易
4. 遵守平台规则，不要发布违规内容
5. 如遇到问题，请及时联系客服
